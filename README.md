# VibeFlow

https://vibe-flow-theta.vercel.app/

1. Register  
2. Explore the sidebar
<img width="405" height="978" alt="image" src="https://github.com/user-attachments/assets/355e6b75-b5ad-4fa0-b1d9-9cfaa2877d33" />

3. Add a new post or reel from the feed section or from the create section  
4. You can comment, like, and share it also  
5. Create another user in incognito, and you can also search for the user from the sidebar search and send a request  
6. Voice messaging is also available  
7. You can join a guild (a public one) or create a private one for your friends  
8. The only way to join a private guild is through an invite link from the owner
 <img width="1859" height="1009" alt="image" src="https://github.com/user-attachments/assets/16530f55-2db0-4c2f-b8ff-3a70fd35d997" />

9. Owner transfer is also available  
10. The reels section has a "watch together" feature—select the link and paste it in your incognito window where you created the second account  
    note: you both have to be following each other for this
    <img width="1848" height="1013" alt="image" src="https://github.com/user-attachments/assets/cd6bbc8e-9192-4634-aa8d-957bb0026e3c" />

11. You can edit the private/public setting of your account on the settings page
